### Для запуска нужно выполнить следующие команды:

- npm install
- npm run dev
- php artisan migrate:fresh --seed

### Для проверки email нужно указать настройки почтового сервера:
- MAIL_MAILER=
- MAIL_HOST=
- MAIL_PORT=
- MAIL_USERNAME=
- MAIL_PASSWORD=
- MAIL_ENCRYPTION=
- MAIL_FROM_ADDRESS=
- MAIL_FROM_NAME=
